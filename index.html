Abstract
This thesis addresses the computational challenges of solving high-dimensional matrix differential equations $\dot{A}(t) = F(A(t))$, where standard integrators fail due to $O(n^2)$ memory constraints. We analyze Randomized Low-Rank Runge-Kutta (RLRK) methods, which constrain the solution to a low-rank manifold $\mathcal{M}_r$ using the generalized Nystr√∂m approximation.
3. Methodology: Randomized Sketching
Instead of a deterministic SVD, we employ randomized sketching. The algorithm constructs a subspace approximation by multiplying the data matrix $A$ with a random test matrix $\Omega$.
Y = A\Omega, \quad \text{where } \Omega \text{ is Gaussian or SRFT}
Key Innovation: Using Subsampled Randomized Fourier Transforms (SRFT) reduces sketching complexity from $O(mn^2)$ to $O(mn \log n)$. Our benchmarks confirm SRFT significantly outperforms dense Gaussian matrices for $n > 8192$.
5. Numerical Experiments
Lyapunov Equation: RLRK4 maintains 4th-order convergence even when Projected RK4 (PRK) collapses due to large projection errors in high-perturbation regimes ($\alpha=1$).
Allen-Cahn Equation: For stiff problems, the fixed-step RLRK4 method proved more stable than the adaptive DOPRI45, which suffered from numerical noise.

def utility_free_control(predicted_mu, current_sigma, history):
    # Utility-free approach maximizing geometric growth
    raw_opt = predicted_mu / (current_sigma**2 + 1e-6)
    
    # Robust Z-Score (Median & IQR)
    w_median = np.median(history[-252:])
    iqr = np.percentile(history, 75) - np.percentile(history, 25)
    z_score = (raw_opt - w_median) / (iqr * 0.74)
    
    # Crisis Mode: Halve risk if daily vol > 2%
    scaling = 0.5 if current_sigma > 0.02 else 0.8
    
    return 1.0 + np.tanh(z_score * scaling)
