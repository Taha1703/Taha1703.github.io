Hull Tactical: Market Prediction
Strategy Paper: Framework, Models & Theory
Team Strategy Analysis ‚Ä¢ February 2026 ‚Ä¢ Taha Atlagh
Abstract: This document serves as the technical specification for the Hull Tactical Market Prediction competition. We analyze volatility constraints and derive mathematical requirements for maximizing the Sharpe Ratio. We utilize LightGBM ensembles and feedforward neural networks under strict time-series cross-validation.
1. Framework & Constraints
The core task is to predict the Excess Returns of the S\&P 500. We challenge the Efficient Market Hypothesis (EMH) by identifying statistical inefficiencies ("Alpha") using ML.
r_excess = r_SPY - r_risk_free
1.1 Volatility Constraint (120%)
The strategy volatility σ_Strat must not exceed 1.2x the market volatility σ_Market. This implies a mandatory Volatility Control Mechanism.
σ_Strat ≤ 1.2 · σ_Market
2. Quantitative Strategies
2.1 Volatility Scaling (Moreira & Muir)
Risk and return are often negatively correlated. To maximize Sharpe Ratio, we scale exposure inversely to variance:
w_t = min(2, c / σ^2_t)
Where w_t is the allocation weight. We use GARCH(1,1) to estimate the variance σ^2_t.
2.2 Fractional Differentiation
Financial data is non-stationary. Instead of integer differencing (d=1), we use fractional d ∈ (0, 1) to preserve memory while achieving stationarity.
3. Model Architecture
We employ an ensemble approach. LightGBM optimizes a differentiable loss function using 2nd order Taylor expansion (Newton's method).
L^(m) ≈ ∑ [g_i h_m(x_i) + 0.5 k_i h_m^2(x_i)]
This is combined with Ridge/Lasso regression for feature selection and robustness against noise.
